# Pobelitely-2team
Хакатон: Транспортная аналитика QGIS
Общая информация

Набор инструментов для анализа транспортной доступности, разработанный в рамках хакатона. Модули предназначены для интеграции в QGIS и решения задач городского планирования.
Установка

    Скопируйте папку с плагином в ~/.local/share/QGIS/QGIS3/profiles/default/python/plugins/

    Запустите QGIS

    В меню выберите: Плагины → Управление и установка плагинов → Установленные

    Активируйте плагин "Хакатон: Транспортная аналитика"

Структура модулей

    Изохроны с рельефом - построение пешеходных зон доступности с учётом высот

    Анализ ОТ - оценка доступности общественного транспорта

    Подсчёт населения - расчёт жителей в зонах доступа

Требования

    QGIS 3.x

    Python 3

    Пространственные данные города (графы, изолинии, здания, остановки)

Использование

    Подготовьте слои данных в QGIS

    Запустите нужный инструмент через меню Processing → Toolbox

    Настройте параметры

    Получите результат на карте и в таблице атрибутов

Поддерживаемые форматы

    Векторные слои: Shapefile, GeoPackage, PostGIS

    Растровые данные: GeoTIFF

    Проекции: любые CRS, поддерживаемые QGIS

Контакты
